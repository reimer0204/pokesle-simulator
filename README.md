# ポケスリシミュレーター

一般ユーザーの方はこちらからご利用ください
[https://reimer0204.github.io/pokesle-simulator/](https://reimer0204.github.io/pokesle-simulator/)

## 特徴
厳選評価とチーム編成のシミュレーションができるツールです。
サーバーレスですべての処理はあなたの端末上で行われます。
Googleスプレッドシートと連携させる機能があるので、複数端末でデータを同期させることも可能です。

## 開発
```
npm run dev
```

## ビルド
```
npm run build
```
